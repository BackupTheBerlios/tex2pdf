Tex2pdf/Lyx2pdf Homepage

Description

tex2pdf is a bash shell script that generates a PDF document from LaTeX or LyX document. The current stable release (named lyx2pdf) can only handle LyX documents as sources. Tex2pdf (current beta release) can process both: LyX and LaTeX documents.

When the input file is a LyX document the script produces a LaTeX file with the LyX export function first and than processes the exported LaTeX document.
LaTeX documents are searched for included files (other LaTeX docs, images), prepared for pdflatex with sed, included EPS/PS/PSTEX images are converted to PDF with epstopdf and finally the script runs pdflatex, bibtex and thumbpdf.

Dependencies

  • LyX
  • pdftex
  • sed
  • epstopdf
