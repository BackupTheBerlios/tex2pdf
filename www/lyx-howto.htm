HOWTO use tex2pdf with LyX

The major task of tex2pdf is to provide an easy way to generate PDF from INSIDE and outside Lyx.

It should handle all the stuff you have to do (e.g. EPS translation, insertation of extra TeX stuff for pdftex, ...).

What you have to do to make tex2pdf work from inside LyX:

Install tex2pdf

  1. Download tex2pdf (I suggest the Perl version, beta7 should be fine) from http://developer.berlios.de/projects/tex2pdf/
  2. Unpack it with 'tar -xfz FILE'
  3. Make sure it is executable with 'chmod a+rx tex2pdf'
  4. Put it in your binary path, (e.g. /usr/local/bin) , e.g. 'mv tex2pdf /usr/local/bin'.
  5. Run 'tex2pdf --configure' ('tex2pdf -r' for bash port) as the regular user who wants to use it.
    NOTE: The script refuses to start without prior configuration!
  6. Have a look at possible command line options with 'tex2pdf --help '

Set up LyX to use tex2pdf

  1. Choose in the following menu: Edit->Preferences->Converters->Converters 'LaTeX->PDF(pdftex)' and set the 'Converter' field to your tex2pdf executable with all it options and the argument '$$i' at the end: e.g.: /usr/local/bin/tex2pdf $$i
  2. Confim your changes. Now it should work.
  3. Report any problems to the tex2pdf list: tex2pdf-devel@lists.berlios.de and/or the lyx-users list.
To use tex2pdf from outside Lyx simply call it by 'tex2pdf [OPTIONS] DOCUMENT.lyx' or 'tex2pdf [OPTIONS] DOCUMENT.tex'
